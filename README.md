# UrlShortener (.NET 8 MVC + EF Core + MySQL)

Готовый минимальный проект под тестовое задание: сокращение URL, таблица ссылок, редактирование, удаление, подсчёт переходов, автоматические миграции при запуске.

## Что сделано
- ASP.NET Core MVC (.NET 8)
- MySQL / MariaDB через Pomelo EF Core provider
- Генерация непредсказуемых short code на базе `RandomNumberGenerator`
- Уникальность short code через проверку в БД
- Автоматический `Database.Migrate()` при старте
- Главная страница с таблицей
- Создание / редактирование / удаление ссылок
- Редирект по короткой ссылке
- Подсчёт количества переходов
- Базовая валидация URL

## Как запустить
1. Установить .NET 8 SDK
2. Поднять MySQL или MariaDB
3. Изменить строку подключения в `appsettings.json`
4. Выполнить:

```bash
dotnet restore
dotnet run
```

При старте приложение само применит миграции через `Database.Migrate()`.

## Важно
В этой среде у меня нет установленного .NET SDK, поэтому я не смог локально собрать и прогнать проект. Но структура, конфигурация и код подготовлены так, чтобы ты мог открыть решение и запустить у себя после настройки строки подключения.
